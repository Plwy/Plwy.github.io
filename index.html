TensorRT 简介

模型部署   DL_Deploy, TensorRT, NVIDIA

概述

TensorRT 是一个前向推理框架。在推理过程中，基于 TensorRT 的应用程序的执行速度可以比 CPU 平台速度快 40 倍。

核心特点

  • 针对 NVIDIA GPU 优化
  • 支持 INT8 和 FP16 量化
  • 动态张量核心支持
  • 层融合优化

硬件兼容性

不同的硬件需要匹配不同的 CUDA 库，然后还需要进行测试，比如选核等操作。

CUDA 基础

TensorRT 以 NVIDIA 的并行编程模型 CUDA 为基础构建而成。

使用方法

安装 TensorRT 需要匹配对应的 CUDA 和 cuDNN 版本，建议使用官方 Docker 镜像。

代码示例

import tensorrt as trt

logger = trt.Logger(trt.Logger.WARNING)
builder = trt.Builder(logger)
config = builder.create_builder_config()

# 创建网络
network = builder.create_network(1 << int(trt.NetworkDefinitionCreationFlag.EXPLICIT_BATCH))
parser = trt.OnnxParser(network, logger)

# 解析 ONNX 模型
with open("model.onnx", "rb") as f:
    parser.parse(f.read())

# 构建引擎
engine = builder.build_engine(network, config)
