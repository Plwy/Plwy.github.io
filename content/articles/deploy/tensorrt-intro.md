# TensorRT 简介

TensorRT 是 NVIDIA 面向推理场景提供的高性能优化框架，适合把训练完成的模型进一步压缩、编译并部署到 GPU 环境中。

## 它解决什么问题

当模型从训练环境进入线上推理阶段后，瓶颈通常不只是算力本身，还包括内存访问、算子融合和批处理策略。TensorRT 的价值，就在于把这些细节向更高性能方向重新组织。

## 核心特点

- 针对 NVIDIA GPU 做了深度优化
- 支持 FP16 与 INT8 量化
- 支持算子融合与层级优化
- 适合和 Triton 等服务框架搭配使用

## 部署时容易忽略的点

真正影响效果的，往往不是能不能跑起来，而是 profile、输入范围、校准数据和观测指标是否一起准备好了。
