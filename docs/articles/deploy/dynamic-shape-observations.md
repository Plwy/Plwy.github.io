# 动态 Shape 的性能观察

**日期**: 2026-03-26
**分类**: 深度学习部署
**标签**: TensorRT, Dynamic Shape, 性能优化

---

## 什么时候会用到动态 Shape

当输入尺寸不固定，或者需要兼容多个分辨率和序列长度时，动态 shape 会让服务更灵活。

## 需要额外关注的点

- Profile 范围不能只图省事写得过宽
- 常见输入区间要单独压测
- 切换 profile 时要观察延迟抖动

## 一个经验

动态 shape 解决的是适配问题，不一定天然带来性能收益。真正影响体验的，往往是主流输入是否落在最优 profile 区间内。
